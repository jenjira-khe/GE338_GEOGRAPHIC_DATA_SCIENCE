## ภารกิจที่ 3: Methodology

### Spectral Indices ที่ใช้

| Index | Band | วิเคราะห์อะไร |
|---|---|---|
| NDVI | (B8-B4)/(B8+B4) | ความหนาแน่นพืชพรรณ |
| NDWI | (B8-B11)/(B8+B11) | ความชื้นในพืช/ดิน |
| MNDWI | (B3-B11)/(B3+B11) | น้ำผิวดิน (>0 = มีน้ำ) |
| EVI | 2.5*(B8-B4)/(B8+6B4-7.5B2+1) | พืชหนาแน่น (ลด saturation) |

### Workflow

```
SHP → GEE FeatureCollection → Sentinel-2 SR → Mask Cloud → Median Composite
    ↓
    ├── NDVI, MNDWI, NDWI, EVI (ทั้ง 2 ช่วง)
    ├── Zonal Statistics รายอำเภอ
    ├── ΔNDVI Change Detection
    ├── NDVI Classification + Area (3 scales สำหรับ Uncertainty)
    └── Scatter NDVI vs MNDWI (2,000 sampled pixels)
    
```

### ผลที่ได้

**Zonal Statistics:** อำเภอที่มี ΔNDVI ติดลบมากคือพื้นที่เกษตรกรรมห่างแหล่งน้ำ อำเภอริมโขงมี ΔNDVI น้อยกว่าเพราะพืชได้รับความชื้นจากดิน stdDev สูงบ่งถึงความหลากหลายการใช้ที่ดิน

**Change Detection:** พื้นที่สีแดง (ΔNDVI < -0.2) กระจุกในพื้นที่เกษตรน้ำฝนที่ไม่มีแหล่งน้ำสำรอง

**NDVI-MNDWI:** R² ลดลงในช่วงที่ 2 — พืชบางส่วนยังมี biomass แต่ดินแห้งแล้ว เป็นสัญญาณเตือน drought stress

## คำถาม

**1. ถ้าเลือกดาวเทียมอีกดวง ผลการวิเคราะห์จะต่างออกไปอย่างไร? ลองทดสอบและเปรียบเทียบ**  
 Landsat 8 (30m, Revisit 16 วัน) จะเห็นรายละเอียดระดับแปลงเกษตรขนาดเล็กน้อยลง ค่า NDVI จะใกล้เคียงกัน แต่ ΔNDVI ในพื้นที่เล็กจะหายไปบางส่วน แต่หมาะสำหรับ Long-term change detection

**2. Index ที่เลือกมีข้อจำกัดอะไร? มี Index อื่นที่เหมาะกว่าสำหรับพื้นที่นี้หรือไม่?**
- NDVI มีจุดอิ่มตัวในป่าหนาแน่น (LAI > 3) — ควรใช้ EVI เสริม
- MNDWI อาจสับสนกับเงาเมฆ
- Index ที่ควรเพิ่ม: LSWI สำหรับติดตามน้ำในดิน เพื่อให้แม่นขึ้น

**3. ถ้าต้องนำผลลัพธ์นี้ไปใช้จริงกับหน่วยงานรัฐ อะไรคือข้อมูลที่ยังขาดอยู่?**
- Ground Truth เพื่อ Validate ผลการจำแนก , ข้อมูลประเภทการใช้ประโยชน์ที่ดิน (LULC) ระดับอำเภอ, ข้อมูลปริมาณน้ำฝนรายเดือน

**4. การเลือก Scale ในการ Export มีผลต่อขนาดไฟล์และความแม่นยำอย่างไร?**  

| Scale | ขนาดไฟล์ | ความแม่นยำ |
|---|---|---|
| 10m | 4× ของ 20m | ไม่เพิ่มสำหรับ Band 20m (interpolated) |
| **20m** | **เหมาะสม** | **ตรงกับ native resolution** |
| 100m | เล็กมาก | ไม่พอสำหรับระดับตำบล |

เลือก 20m เพราะเป็น Native Resolution ของ SWIR Band และสมดุลระหว่างความแม่นยำกับขนาดไฟล์

## สรุปผล
| ประเด็น | ฤดูแล้ง | ฤดูฝน | การเปลี่ยนแปลง |
|---|---|---|---|
| NDVI เฉลี่ย | ~0.35–0.45 | ~0.50–0.65 | +0.15–0.20 |
| พื้นที่น้ำ | น้อยกว่า | มากกว่า | เพิ่มขึ้น |
| MNDWI ริมโขง | ต่ำ | สูง | ชัดเจน |

NDVI เพิ่มขึ้นในฤดูฝน — พื้นที่เกษตรกรรมและป่าไม้ฟื้นตัวหลังได้รับน้ำฝน
MNDWI สูงตามแนวแม่น้ำโขง — แม่น้ำโขงมีระดับน้ำสูงขึ้นในฤดูฝน ท่วมพื้นที่ริมตลิ่ง
อำเภอที่ติดแม่น้ำโขง (เมืองหนองคาย, ท่าบ่อ, ศรีเชียงใหม่) มี MNDWI สูงกว่าอำเภอไกลจากแม่น้ำโขง
ความสัมพันธ์ NDVI-MNDWI: ในฤดูฝน มีความสัมพันธ์เชิงบวกสูงกว่า แสดงว่าพื้นที่ชื้นมีพืชพรรณมากกว่า
